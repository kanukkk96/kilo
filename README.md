Kilo
===

Kilo is a small text editor in less than 1K lines of code (counted with cloc).

Usage: kilo `<filename>`

Keys:

    CTRL-S: Save
    CTRL-Q: Quit
    CTRL-F: Find string in file
    CTRL-X: Remove one line
    CTRL-O: Help Option
    CTRL-N: Display row number
    CTRL-D: Move to end of current line
